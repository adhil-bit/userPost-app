

# UserPost App

This app allows you to create and manage posts and users using MongoDB with Mongoose.

## Quick Start

### Step 1: Clone the Repository

git clone https://github.com/adhil-bit/userPost-app.git

### Step 2: npm install

### step 3: node app.js
